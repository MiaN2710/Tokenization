## Content update in Kinethic website:

| Page/Folder to Update | Details | File/Folder |
| :--- | :--- | :--- |
| Homepage | To update contents of the homepage, header, header dropdown, and navigation links. | hugo.yaml |
| Pages | General pages such as Mission, Networks, Contact, etc. | content folder |
| Post-Pages | Pages created specifically for published articles or blog posts. | content/posts folder |
| Images (General) | Images that are referenced within posts or pages. You can create subfolders here for organization. | themes/tailbliss/assets/images folder |
| Logo/Favicon Images (Fast Loading) | Images like logos and favicons that require faster loading and direct access. | themes/tailbliss/static/images folder |
| Favicon Images (Specific) | Favicon images that require faster loading and direct access. | themes/tailbliss/static/favicon folder |
